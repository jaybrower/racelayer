## Internal

- **Vitest test framework wired up.** New `npm test` / `npm run test:watch` scripts run Vitest against `tests/**/*.test.ts`. Scope is pure logic only — no Electron, no React rendering, no jsdom. Manual scenarios live in `docs/test-plan.md` (added separately).
- **`Test` GitHub Action** runs `npm test` on every PR and on pushes to `main` / `release/v*`, so a broken pure-logic change can't merge silently.
- **Pure logic extracted into `lib.ts` modules** alongside each overlay's component (`PitStrategy/lib.ts`, `Relative/lib.ts`, `Gauges/lib.ts`). The component file now imports the formatter / regression / stint-detection / fuel-stat functions instead of defining them inline — same behaviour, but the logic is now unit-testable without bringing React along.
- **Initial test coverage** (85 unit tests):
  - Stint detection, tire-deg trend window, stint-best delta, fuel-stat fallbacks
  - Closing-rate least-squares regression (samples, time span, outlier dampening, sign conventions for car-ahead vs car-behind)
  - Relative-gap track-position wrap, reference-lap-time fallback chain, iR-change Elo approximation
  - Lap-time / delta / gap / fuel formatters
  - `mergeWithDefaults` forward-compat (old configs, missing fields, malformed branches)
