# v0.1.0

_Released 2026-05-14 · initial release_

The first public build of RaceLayer — a lightweight, transparent iRacing HUD overlay for Windows. Reads telemetry directly from iRacing's shared memory via koffi FFI (no DLL injection or sidecar process needed) and renders each overlay as its own transparent, always-on-top Electron window.

## Added

- **Relative overlay** — Proximity list centered on the player car, showing drivers immediately ahead and behind.
  - Gap in seconds (tenths) or laps
  - Current position and position gained/lost vs. starting grid
  - iRating and color-coded Safety Rating badge (red → amber → green → blue based on sub-rating)
  - Estimated iRating change based on current race positions
- **Gauges overlay** — Full-width instrument bar.
  - RPM bar with gradient colour shift and redline indicator
  - Throttle and brake input trace
  - Gear, speed (MPH), lap delta to personal best
  - Fuel level with laps remaining
  - Traction Control and ABS indicators showing dial level and flashing when actively intervening (auto-hidden on cars that don't expose these)
- **Pit Strategy overlay**
  - Fuel level, per-lap consumption (measured at lap crossings, not instantaneous), and laps remaining
  - Tire degradation: session-best lap as a rolling baseline, up to 3 recent wear laps with delta, and a headline average-deg number
  - Estimated pit window — last lap you can safely pit on current fuel
- **Tire Temps overlay** — Four-corner display with colour-coded heat mapping across each tire's inner / middle / outer zones. Surface temps when available, carcass temps as fallback (auto-hidden when the car exposes neither).
- **Per-session-type visibility** — every overlay and element independently toggleable for practice / qualifying / race
- **Auto-hide unsupported elements** — TC, ABS indicators, and Tire Temps hide automatically when the current car doesn't support them
- **Draggable layout mode** — `Ctrl+Shift+L` unlocks all overlays for repositioning; positions persist per monitor configuration
- **Launch on startup** — optional Windows login item so RaceLayer is ready before iRacing opens
- **Configurable global shortcuts** — remap layout mode and settings to any key combo
- **Dev Mode** — preview overlays with simulated mock data when iRacing isn't running

## Install

- `RaceLayer-0.1.0.exe` — NSIS installer with Start Menu entry and clean uninstall
- `RaceLayer-portable-0.1.0.exe` — single-file portable build

**Requirements:** Windows 10 or later (64-bit), iRacing
