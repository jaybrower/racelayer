# v0.1.1

_Released 2026-05-14_

## Added

- **In-app updates** — RaceLayer can now check for, download, and install updates without leaving the app. Open Settings to find the new **Updates** section at the top.
  - Click **Check for updates** to see if a newer version is available
  - When one is found, click **Download** — a progress bar shows the download as it completes
  - Once downloaded, click **Restart & Install** to apply immediately, or just quit normally and it will install on next launch
  - The current installed version is always displayed so you know exactly what you're running
  - Powered by `electron-updater` with the GitHub provider; consumes the `latest.yml` manifest published with each stable release

## Install

- `RaceLayer-0.1.1.exe` — NSIS installer
- `RaceLayer-portable-0.1.1.exe` — single-file portable build
- Already on 0.1.0? You'll need to install this build manually — the new updater only ships *with* 0.1.1, so 0.1.0 doesn't know how to detect it
